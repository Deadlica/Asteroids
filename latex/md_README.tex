This is a game inspired by the classical retro game Asteroids. It\textquotesingle{}s built in c++ using the external library SFML\hypertarget{md_README_autotoc_md1}{}\doxysection{Setup}\label{md_README_autotoc_md1}
It can be run either through Cmake, or using a regular make command. For regular makefile command type the following in the terminal


\begin{DoxyCode}{0}
\DoxyCodeLine{make}

\end{DoxyCode}


Once it\textquotesingle{}s compiled, you\textquotesingle{}re good to go to launch throught the terminal with


\begin{DoxyCode}{0}
\DoxyCodeLine{./main}

\end{DoxyCode}
\hypertarget{md_README_autotoc_md2}{}\doxysection{Gamemodes}\label{md_README_autotoc_md2}
The game features 2 different gamemodes. First is very similar to the classical asteroids gamemode where you are supposed to shoot as many asteroids as possible to rack up points. The second gamemode is a little boss fight. Which features an enemy spaceship that will try to defeat you before you can defeat it.
